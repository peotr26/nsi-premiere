Les systèmes *NIX, ou dit de type UNIX, est une famille de systèmes d’exploitations (OS). Le premier OS de cette famille est UNIX développé par Bell Labs. Cette famille se divise en 3 sous-familles:

  1. UNIX génétique
  2. UNIX commercial
  3. UNIX fonctionnel

UNIX génétique

Les systèmes d’exploitations de type UNIX génétique contiennent sois le code source d’UNIX ou sont basé dessus. On y retrouve les UNIX dit commerciaux comme Solaris de Sun Microsystems ou AIX d’IBM mais aussi les systèmes d’exploitations développé par les universités comme BSD de l’Université de Californie et tout ces dérivés.

UNIX commercial

Les systèmes d’exploitations de type UNIX commercial sont des systèmes d’exploitation qui sont certifié par l’Open Group. On y retrouve par exemple MacOS (anciennement OS X) qui est lui basé sur FreeBSD et NeXTStep.

UNIX fonctionnel

Les systèmes d’exploitations de type UNIX fonctionnel sont ceux qui ne respecte pas proprement les spécifications UNIX mais dont le fonctionnement est extrêmement proche. On y retrouve notamment Linux et Minix (OS qui a servi d’inspiration à Linus Torvalds pour Linux).

Utilisation

De nos jours, les systèmes de type UNIX sont très populaires dans les serveurs, mainframes et super-ordinateurs avec notamment Linux et FreeBSD pour les serveurs et super-ordinateurs, et AIX et z/OS pour les mainframes. Mais ces derniers sont aussi populaire avec les développeurs avec MacOS et Linux. Ainsi Apple est plus important fabricant de station de travail UNIX.
Il est à noté que Android (l’OS le plus répandu dans le marché des smartphones) est basé sur une version modifié de Linux mais n’est pas considéré comme étant de type UNIX car l’espace utilisateur s’éloigne trop des spécifications.

Arborescence des systèmes de type UNIX

Windows et UNIX

Dans les années 90 avec Windows NT 4, Microsoft rend Windows compatible POSIX (norme inspiré d’UNIX). À partir de 2016, Windows intègre WSL, un sous-système Linux qui permet d’exécuter des programmes conçu pour Linux. Red Hat et Cygnus, crée en 1995, Cygwin, un programme permettant d’avoir accès à un environnent UNIX dans Windows.

Liste des plus important systèmes de type UNIX

  • Research UNIX (l’original) (Bell Labs)
  • Linux (UNIX fonctionnel) (Linus Torvalds)
  • MacOS (UNIX commercial) (Apple)
  • FreeBSD (UNIX génétique) (The FreeBSD Project)
  • Solaris (UNIX génétique et commercial) (Sun Microsystems puis Oracle)
  • AIX (UNIX génétique et commercial) (IBM)
  • Minix (UNIX fonctionnel) (Andrew S. Tanenbaum)
  • NetBSD (UNIX génétique) (The NetBSD Foundation)
  • OpenBSD (UNIX génétique) (Theo de Raadt)
  • HP-UX (UNIX génétique et commercial) (HP)
